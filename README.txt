Para trocar entre os modos 2D e 3D pressione a barra de espaço

MODO 3D:
Para aumentar e diminuir a altura das figuras utilize as setas para cima e para baixo respectivamente
Para controlar o ZOOM utiliza a roda do mouse
Para controlar a rotação utilize o botão esquerdo e para controlar o pan utilize o botão direito do mouse

Para modificar as texturas utilize o menu  (é possivel esconde-lo apertando 'h')
após escolher uma textura mais oções de modificação ficarão disponíveis
